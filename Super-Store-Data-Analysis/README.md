# 📊 Super Store Data Analysis Dashboard

## 📌 Project Overview

The **Super Store Data Analysis Dashboard** is an interactive Business Intelligence dashboard developed using **Power BI** and the **Super Store Sales Dataset**. The dashboard provides comprehensive insights into sales performance, profit trends, customer segments, product categories, shipping modes, and regional sales distribution.

The objective of this project is to transform raw sales data into meaningful visual insights that help businesses monitor performance, identify trends, and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance.
* Monitor monthly sales and profit trends.
* Identify top-performing product categories and sub-categories.
* Compare sales across different customer segments.
* Evaluate regional and state-wise sales performance.
* Analyze shipping mode preferences.
* Build an interactive dashboard for business users.

## 💡 Key Insights

* Consumer customers contribute the highest share of total sales.
* The West region records the highest sales performance.
* Standard Class is the most frequently used shipping mode.
* Office Supplies generate the highest sales among categories.
* Phones are the top-performing sub-category.
* September shows the highest monthly sales.
* California contributes the highest state-wise sales.
